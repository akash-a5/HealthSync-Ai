🏥 HealthSync AI

AI-Powered Smart Healthcare & Supply Chain Management for Primary Health Centers

HealthSync AI is an AI-powered healthcare management platform built for Primary Health Centers (PHCs) and Community Health Centers (CHCs). It enables healthcare staff to efficiently manage medicines, beds, doctors, patients, and inter-center resource sharing while leveraging Google Gemini AI for intelligent predictions and decision support.

Built for the Google Build with AI Hackathon.

⸻

🌟 Problem Statement

Healthcare centers often face:

* Medicine shortages
* Expired medicine wastage
* Poor bed utilization
* Manual patient records
* Inefficient doctor scheduling
* Lack of coordination between nearby PHCs
* No AI-powered forecasting

HealthSync AI solves these challenges using real-time analytics and Google Gemini AI.

🚀 Installation

Clone the repository

git clone https://github.com/YOUR_USERNAME/HealthSync-AI.git

Move into the project

cd HealthSync-AI

Install dependencies

npm install

Create a .env

GEMINI_API_KEY=YOUR_API_KEY

Run the development server

npm run dev

Open

http://localhost:3001

⸻

📦 Production Build

Build

npm run build

Start production

npm start
